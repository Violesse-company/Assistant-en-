🏋️‍♂️ Powerlifting Weightlifting Game Description

🎮 Game Overview

This game is a weight calculation training game that simulates the weight adjustments made by "assistants" during powerlifting competitions.
You can practice quickly determining the correct plate configuration while understanding the rules.

💾 Recording Mode Flow

  • First, look at the weight and think about the plates to attach.
  • Press the plates to accurately set the weight.
  • Be sure to attach the official color (2.5kg) at the end of the plates.
  • If correct, it will be displayed as correct.
  • If incorrect, it will be displayed as over.
  • If the color is not attached, it will prompt you to attach 2.5kg.
  • Once the plates are correctly arranged, proceed to the next question by clicking "Next."
  • After 5 rounds, the results will be automatically tallied.
  • To close, press the close button.

📊 Results Summary

  • Number of times plates were correctly attached (out of 5)
  • Average time taken to reach the correct answer (seconds)
  • Correct answer rate (%)
  • Average speed (reaction time)

💪 Practice Mode

  • Press the OK button to change the target weight.
  • The judgment of correct/incorrect will be displayed on the spot.
  • Scores and times are not recorded.
  • It is useful for practicing the sense of color placement and plate composition.

💡 Plate Placement Rules

  • Weight range: 25kg to 300kg.
  • Available plates: 25 / 20 / 15 / 10 / 5 / 2.5 / 1.25 kg.
  • Always attach the color plate (2.5kg) at the end of the plates.
  • Attaching multiple color plates in between is not allowed.
  • Placing 1.25kg after a color plate is also not allowed.
